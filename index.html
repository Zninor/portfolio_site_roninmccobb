My work

FairShare

FaireShare is a native application designed to help groups of users add multiple purchases to an overall bill and divide the cost amongst all users evenly with little hassle. With this app, users can individually pay for big purchases like restaurants, rideshare trips, and shared living costs and know they will only pay their fair share. For this project, I created a series of wireframes that went through many iterations and user tests before developing a style guide and user flow.

Baked

Baked is a responsive web app that shows users food recipes and how to include THC in varying amounts. Users are walked through how to create THC infused cooking ingredients, and then shown how much they should add to a recipe in order to achieve the desired potency. For this project, I began with user interviews to create user personas, and then developed a set of mockups after several prototype and A/B tests to see what users preferred.
